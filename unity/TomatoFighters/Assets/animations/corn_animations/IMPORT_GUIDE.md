## Import Steps

### 1. Copy Files
Copy the entire `corn_animations/` directory into your Unity project under `Assets/Sprites/corn/`.

### 2. Configure Sprite Import Settings
For each spritesheet PNG:
1. Select the file in Unity's Project window
2. Set **Texture Type** → Sprite (2D and UI)
3. Set **Sprite Mode** → Multiple
4. Set **Pixels Per Unit** → 128
5. Set **Filter Mode** → bilinear
6. Open **Sprite Editor** and slice using grid:
   - Cell Size: 128 x 256
   - Pivot: Bottom Center
7. Click **Apply**

## Troubleshooting

- **Sprites look blurry**: Set Filter Mode to Point (no filter) and disable compression
- **Animation too fast/slow**: Adjust Sample Rate in Animation window (default: 12 fps)
- **Transparent edges visible**: Ensure sprite mesh type is set to Tight, not Full Rect
